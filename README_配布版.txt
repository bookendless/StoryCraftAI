=== AIストーリービルダー ローカル版 v1.0.0 ===

【クイックスタート】
1. Node.js をインストール → https://nodejs.org/
2. start-local.bat をダブルクリック
3. ブラウザが自動で開きます（http://localhost:5000）

【必要なソフトウェア】
✓ Node.js 18.x以上（必須）
✓ Ollama（AI機能用、オプション）→ https://ollama.ai/

【初回セットアップ】
- 依存関係の自動インストール
- アプリケーションの自動ビルド  
- メモリストレージの初期化
- サンプルプロジェクトの作成

【データ保存について】
⚠️ 現在はメモリストレージ使用のため、アプリ終了時にデータは消失します
→ 将来のアップデートでファイル保存機能を追加予定

【AI機能について】
・基本補完機能：簡易的な文章補完
・Ollama連携：ローカルAIでより高度な補完
　※ Ollamaインストール後 "ollama pull llama3.2:3b" でモデル追加

【トラブル解決】
Q: バッチファイルがすぐ閉じる
A: コマンドプロンプトで "node server/index.local.cjs" を実行

Q: 「依存関係インストールエラー」
A: 管理者権限でstart-local.batを実行

Q: 「ポートが使用されています」
A: ブラウザを閉じて再実行（自動で別ポート使用）

Q: Node.jsエラー
A: Node.js 18.x以上をインストール・再起動

【使い方】
1. 新プロジェクト作成
2. キャラクター設定（AIボタンで補完）
3. プロット構成（AI提案機能）
4. あらすじ・章・エピソード作成

【システム要件】
- Windows 11/10
- メモリ 4GB以上
- Node.js 18.x以上
- 空き容量 2GB以上
